# pensamento_computacional_projeto-Barbearia
Este é um protótipo funcional desenvolvido em Python para automatizar o fluxo de atendimento de um salão de beleza ou barbearia. O projeto foca na experiência do usuário via terminal, permitindo o controle de agendamentos e serviços de forma direta.

🚀 Funcionalidades Atuais O sistema oferece um menu interativo com as seguintes operações:

Agendamento de Clientes: Captura de dados básicos (Nome e Telefone) para início do atendimento.

Gestão de Horários: Organização da agenda definindo horários e períodos específicos.

Alteração de Serviço: Flexibilidade para mudar o tipo de atendimento (Corte, Barba ou Combo) após o cadastro.

Cancelamento Eficiente: Remoção de agendamentos do sistema de forma rápida.

Relatório de Serviços: Módulo para visualização da produção diária (em desenvolvimento).

🛠️ Tecnologias e Lógica Aplicada Para garantir um código limpo e funcional, foram utilizados os seguintes conceitos:

Laços de Repetição (while True): Mantém o sistema rodando continuamente até que o usuário decida sair.

Entrada de Dados (input): Interação em tempo real para coleta de informações dos clientes.

Estruturas Condicionais (if/elif): Direcionamento do fluxo do programa com base na escolha do usuário.

Saída Formatada (print): Feedback visual imediato para cada ação realizada (confirmações de sucesso).
